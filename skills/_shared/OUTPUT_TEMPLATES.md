## Citation discipline (shared)

- Every factual claim should be traceable to a source.
- Prefer primary sources; if using secondary sources, state that explicitly.
- Maintain a citations list with stable identifiers:
  - `arxiv:<id>`
  - `url:<host>/<path>`
  - `onchain:<chain>:<tx|address|contract>`
  - `coldvision:<resource>/<id>`

## Embedded charts (shared)

When you include charts in Markdown:

- Prefer Mermaid for simple diagrams.
- For numeric plots, include a small “data table” block (CSV) and a “rendering hint”:
  - x-axis field, y-axis field(s), aggregation, time window.
